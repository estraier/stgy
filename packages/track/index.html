STGY Track Viewer Test

これまでに実装した全機能（カラー指定含む）の挙動を確認するためのテストページです。

1. カラフルなピン ＆ オートフィット

data-color 属性によるピンの色の変更テストです。ズーム指定がないためオートフィットが働きます。

2. ポップアップのパーセンテージ指定 ＆ 縦長画像スクロール

data-popup-width と data-popup-height をパーセンテージ(1〜99)で指定します。 縦幅を制限しているため、ポップアップ内でスクロールバーが出ます。（ズームは15で固定）

3. 海外の地図 (CyclOSM デフォルト) ＆ サイズ指定なし(フォールバック)

日本国外のため自動的に CyclOSM がデフォルトレイヤーになります。 親要素に height や aspect-ratio を指定していないため、CSSの min-height: 300px; が適用されます。
