STGY Track Viewer Feature Demo

This page demonstrates the currently implemented STGY Track Viewer features.

0. Minimal data-src loading

This is the smallest useful setup. The map reads GeoJSON directly from data-src.

0.1. Gzipped GeoJSON data URL

This demo loads a gzipped GeoJSON payload from a data:application/geo+json+gzip;base64,... URL.

1. Inline pins and popup sizing

This demo has no explicit data-lat, data-lon, or data-zoom. The viewer fits all inline pins automatically.

2. Fixed view

This demo explicitly sets data-lat, data-lon, and data-zoom. The map keeps the author's chosen view.

3. Route and GeoJSON point hybrid

This demo mixes inline HTML pins and GeoJSON features. It also uses only data-zoom, so the center is computed from the data while the zoom is fixed.

4. Guide pin with data-render="pin"

A source link with data-render="pin" is initially rendered as a guide marker. Clicking the guide marker toggles the corresponding route layer.

5. Initial view from route bounds

This demo has no inline pins and no explicit view. The initial map view is computed from all route coordinates.

6. coordinateProperties overlay and graph

Hover over the route or graph. The map overlay, coordinate marker, and graph readout all refer to the same nearest coordinate sample.

7. Overseas map auto switching

This point is outside Japan, so the viewer should omit GSI base layers and use overseas-friendly base maps.

8. Error: unsupported MIME type

This demo intentionally uses text/plain. The viewer should show a visible unsupported MIME error.

9. Error: broken JSON

The MIME type is valid, but the JSON payload is intentionally broken.

10. Error: missing template

This demo references a missing template ID.

11. Multiple popup links and images

This demo renders multiple safe links and images from GeoJSON properties.

12. Display toggles

These maps use the same route data while changing data-show-overlay and data-show-graph.

12.1. Overlay disabled, graph enabled

12.2. Overlay enabled, graph disabled

12.3. Overlay and graph disabled

13. Multiple route graph switching

This demo contains two LineString routes with different coordinate properties. The first route is selected by default. Click the other route on the map to switch the graph target. Hover over the graph after switching to confirm that the readout, overlay, and coordinate marker use the selected route.
