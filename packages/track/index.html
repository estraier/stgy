STGY Track Viewer Feature Demo

これまでに実装された全機能の挙動を確認するテストページです。

0. 最小構成：data-src だけで読み込む

data-src だけを指定した最小構成のデモです。
TrackJSON / GeoJSON の読み込み元を figure に直接書けることを確認します。

1. 基本機能：ピン描画とポップアップサイズ指定（完全オートフィット）

data-lat, data-lon, data-zoom をすべて指定していません。
自動的に50pxの余白を持って全体が収まるようにズーム計算されます。

2. 画角の完全固定（オートフィット無効）

data-lat, data-lon, data-zoom をすべて指定しています。
著者の意図を完全に尊重するため、マップは動きません。

3. ルートとGeoJSONピン（中心移動＋ズーム維持）

data-zoom="12" のみを指定しています。
この地図には、HTML側のピン（東京駅/紫）と、GeoJSON側のピン（浅草寺/金）の両方が混在して表示されます。どちらもクリックしてポップアップが開きます。

4. ガイドマップ：data-render="pin"

.stgy-track-sources a.track-source[data-render="pin"] のデモです。
最初は代表点だけがピンとして表示され、クリックすると対応するルート表示がトグルされます。
